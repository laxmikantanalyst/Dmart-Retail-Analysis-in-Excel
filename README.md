# 🛒 D-Mart Retail Sales Dashboard (Excel)

## 📌 Project Overview

This project is an interactive **D-Mart Retail Sales Dashboard** developed in **Microsoft Excel** to analyze retail sales performance and generate actionable business insights.

The dashboard enables users to monitor sales, profit, orders, product performance, category contribution, payment trends, and regional profitability through interactive visualizations and slicers.

## 🎯 Business Objectives

- Monitor overall sales performance.
- Analyze profit across different regions.
- Identify top-selling products.
- Track monthly sales trends.
- Understand customer payment preferences.
- Support data-driven business decisions.

## 📈 Dashboard Components

- UPI

### Region-wise Profit Analysis

- Central
- East

- South
- West

Tracks the highest revenue-generating products.
